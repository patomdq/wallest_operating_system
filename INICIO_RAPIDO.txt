┌─────────────────────────────────────────────────────────────────────────┐
│  🚀 PASOS PARA INICIAR                                                  │
└─────────────────────────────────────────────────────────────────────────┘

PASO 1: Instalar Node.js
   → Descargar desde: https://nodejs.org
   → Instalar la versión LTS (Long Term Support)
   → Reiniciar la terminal después de instalar

PASO 2: Configurar Supabase
   → Ir a: https://supabase.com
   → Crear cuenta gratuita (si no tienes)
   → Crear nuevo proyecto
   → Ir a Settings → API
   → Copiar: Project URL y anon public key

PASO 3: Configurar Variables de Entorno
   → Copiar archivo: .env.local.example → .env.local
   → Editar .env.local con tus credenciales:
     NEXT_PUBLIC_SUPABASE_URL=tu_url_aqui
     NEXT_PUBLIC_SUPABASE_ANON_KEY=tu_key_aqui

PASO 4: Crear Tablas en Supabase
   → En Supabase, ir a: SQL Editor
   → Abrir archivo: scripts/migrations.sql
   → Copiar TODO el contenido
   → Pegar en SQL Editor
   → Ejecutar (Run o F5)
   → Verificar que se crearon 13 tablas

PASO 5: Instalar Dependencias
   → Abrir terminal en la carpeta del proyecto
   → Ejecutar: npm install
   → Esperar 2-3 minutos

PASO 6: Iniciar la Aplicación
   
   OPCIÓN A (Windows - Automático):
   → Ejecutar: .\start.ps1
   
   OPCIÓN B (Manual):
   → Ejecutar: npm run dev

PASO 7: Abrir en Navegador
   → Ir a: http://localhost:3000
   → ¡Listo! Verás el Dashboard de WOS 1.0

┌─────────────────────────────────────────────────────────────────────────┐
│  🔧 SOLUCIÓN RÁPIDA DE PROBLEMAS                                        │
└─────────────────────────────────────────────────────────────────────────┘

❌ "Node.js no encontrado"
   → Instalar Node.js y reiniciar terminal

❌ "Cannot connect to Supabase"
   → Verificar credenciales en .env.local
   → Asegurar que no hay espacios extra

❌ "Table doesn't exist"
   → Ejecutar migrations.sql en Supabase SQL Editor

❌ "Puerto 3000 ocupado"
   → Usar otro puerto: npm run dev -- -p 3001
